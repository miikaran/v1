## How to use it?

 #### 1. Clone the repository -
   ```sh
   git clone https://github.com/miikaran/v1.git
  ```
 #### 2. Install npm packages.
   ```sh
   npm install
   ```

 #### 3. Start the application.

  ```sh
  npm run start
 ```
